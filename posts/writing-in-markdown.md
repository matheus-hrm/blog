# Writing in Markdown

Markdown lets you write using simple symbols:

1. Headings with `#`.
2. Lists with `-` or `1.`.
3. **Bold** and *italic* text.
4. Inline `code` and code blocks.

```js
console.log("Markdown inside the terminal!");
```

---

```ts
export type UUID = string;
const posts: Posts = (id: number): Promise<Post> => {}
```

```c
#include <stdio.h>

int main(void) {
    printf("Hello World!\n");
    return 0;
}
```
